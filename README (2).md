#  Mercado Center 

##  Descrição do Sistema
- Apresenta a disponibilidade de produtos de boa qualidade e de preço acessível.
- O uso desse site foi para principalmente para os clientes insatisfeitos com os mercados de sua região.
- Esse mercado vai resolver os problemas como o alto preço de produtos, virando algo acessível a muitos clientes.
- Fortalecimento do comércio local e relações de consumo ético.
- Melhora na saúde pública e bem-estar coletivo.
---

##  Tecnologias Utilizadas
O projeto foi construído utilizando as seguintes tecnologias:
* **Linguagem de Programação:** C++
* **Framework:** .NET (via C++/CLI)
* **Banco de Dados:** SQLite (leve, rápido e local)
* **Ferramenta Adicional:** Typora (para edição e validação de documentos de texto e Markdown)

---

##  Como Executar o Projeto
Para rodar o sistema localmente na sua máquina, abra o terminal do seu computador e execute os seguintes comandos em ordem:

```bash
# 1. Clonar o repositório do mercado
git clone [https://github.com/empresa-ficticia/mercado-center.git](https://github.com/empresa-ficticia/mercado-center.git)

# 2. Entrar na pasta do projeto
cd mercado-center

# 3. Compilar o sistema em C++
g++ src/main.cpp -o MercadoCenter.exe

# 4. Executar o aplicativo compilado
./MercadoCenter.exe#  Mercado Center 

#  Mercado Center 
```
